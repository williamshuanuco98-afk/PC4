# Portal Tecnológico: "Technova Summit 2026"
### Evaluación Final - Hojas de Estilo en Cascada Avanzado (100000SI58)

Este proyecto consiste en el desarrollo del portal integral para un congreso internacional de tecnología, realizado para el **Ejercicio 1: BOOTSTRAP**. La propuesta demuestra el uso avanzado de la grilla responsiva de Bootstrap, personalización visual mediante SCSS y componentes dinámicos.

---

## 🚀 Instrucciones de Ejecución

El proyecto es estático. Puedes abrir `index.html` en tu navegador directamente o usar Live Server.

---

## 📁 Estructura de Archivos
*   `index.html`: Inicio con hero, countdown y beneficios.
*   `agenda.html`: Cronograma por días con tabs y tablas responsivas apiladas.
*   `ponentes.html`: Catálogo de ponentes con enlaces y modales descriptivos.
*   `entradas.html`: Comparación de planes y acordeón FAQ de bootstrap.
*   `inscripcion.html`: Formulario con validación visual dinámica de bootstrap.
*   `contacto.html`: Formulario, mapa referencial y canales de atención.
*   `css/`: Archivos `custom.scss` (SCSS original con mixins y variables) y `custom.css` (estilos compilados).
*   `js/main.js`: Lógica de countdown y toasts.

---

## 🛠️ Tecnologías Aplicadas
*   **Grilla Avanzada de Bootstrap**: Uso de clases responsivas `col-md-*`, `col-lg-*` y adaptabilidad de layouts.
*   **Componentes Bootstrap**: Navbar con Offcanvas, Cards, Accordion, Pills, Modals y Toasts.
*   **Personalización Visual**: Modificación profunda de colores y estilos en [custom.scss](file:///c:/Users/LAB-USR-LNORTE/Desktop/pc4/ejercicio1_bootstrap/css/custom.scss).

# Portal Médico: "Salud 360"
### Evaluación Final - Hojas de Estilo en Cascada Avanzado (100000SI58)

Este proyecto consiste en el desarrollo del portal web completo para una red de centros de salud **"Salud 360"**, realizado para el **Ejercicio 3: MATERIALIZE**. La propuesta demuestra el uso de componentes Material Design personalizados para evitar la visualización predeterminada de la librería.

---

## 🚀 Instrucciones de Ejecución

El proyecto es estático. Puedes abrir `index.html` directamente en tu navegador.

---

## 📁 Estructura de Archivos
*   `index.html`: Servicios principales, especialidades destacadas y acceso a reservas.
*   `especialidades.html`: Clasificación clínica, quirúrgica y de diagnóstico mediante Tabs.
*   `directorio.html`: Directorio con buscador dinámico, tarjetas de especialistas y modales.
*   `reserva.html`: Formulario con paso a paso interactivo, selectores, datepicker y timepicker.
*   `citas.html`: Historial y estados de citas en una tabla responsiva.
*   `contacto.html`: Formulario, horarios, información de sedes y acordeones colapsables de FAQ.
*   `css/custom.css`: Archivo de personalización de variables de color de marca de salud.
*   `js/main.js`: Inicialización de todos los disparadores JS de Materialize y alertas toast.

---

## 🛠️ Tecnologías Aplicadas
*   **Material Design Components**: Sidenav, datepicker, timepicker, modals, collapsibles, selects y toasts.
*   **Diseño Sostenible y Responsivo**: Reajuste elástico y remaquetación de tablas para teléfonos móviles.
